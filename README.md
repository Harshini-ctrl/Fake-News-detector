# How I Built a Fake News Detector | End to End Machine Learning Project



Ever wondered how Artificial Intelligence can analyze News Articles and detect possible Misinformation or Fake news in real time?  
I’ll show you how I built an AI-Powered Fake News Detection System using NLP, Machine Learning, and Large Language Models (LLMs) — a complete real-world AI project. 🚀

We’ll go step by step:
✅ Fetching live news articles using APIs
✅ Summarizing articles with AI
✅ Performing Sentiment Analysis using LLMs
✅ Estimating Fake News Probability
✅ Detecting trending topics and keywords
✅ Grouping similar news using TF-IDF + KMeans Clustering

This is a full End-to-End NLP + Machine Learning Project — perfect for students, developers, and anyone interested in building practical AI systems for real-world applications.

✨ By the end, you’ll learn how to:
• Build a real-time AI news intelligence engine
• Use Large Language Models for NLP tasks
• Apply TF-IDF vectorization and KMeans clustering
• Design scalable AI pipelines using FastAPI and async processing
• Transform raw news data into meaningful insights

📌 Technologies Used:
Python, FastAPI, OpenAI/Groq API, NLP, TF-IDF, KMeans Clustering, AsyncIO, HTTPX, Scikit-learn, Pandas
